Qual é o Teorema de Pitágoras?
O Teorema de Pitágoras estabelece que o quadrado da hipotenusa de um triângulo retângulo é igual à soma dos quadrados dos catetos. (a² + b² = c²)
O que é a hipotenusa?
A hipotenusa é o lado oposto ao ângulo reto em um triângulo retângulo e é o lado mais longo.
Como calcular a hipotenusa em um triângulo retângulo?
Para calcular a hipotenusa, usamos a fórmula c = √(a² + b²), onde a e b são os catetos.
O que são os catetos?
Os catetos são os dois lados que formam o ângulo reto em um triângulo retângulo.
Qual é o valor de c se a = 3 e b = 4?
c = √(3² + 4²) = √(9 + 16) = √25 = 5
Qual a relação entre os lados de um triângulo retângulo 3-4-5?
Este é um triângulo retângulo conhecido como um triplo pitagórico, onde a² + b² = c². Neste caso, 3² + 4² = 5².
O que é um triplo pitagórico?
Um triplo pitagórico é um conjunto de três números inteiros (a, b, c) que satisfazem a equação a² + b² = c².
Como aplicar o Teorema de Pitágoras na vida real?
O Teorema de Pitágoras pode ser usado para calcular distâncias em situações do dia-a-dia, como encontrar a altura de um edifício usando um ângulo de visão.
Quem foi Pitágoras?
Pitágoras foi um matemático e filósofo grego, conhecido principalmente pelo teorema que leva seu nome, que descreve a relação entre os lados de um triângulo retângulo.
